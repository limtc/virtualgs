Fun and Games

 
 

Programming is supposed to be fun. It is supposed to be as easy as sitting down, type a few commands, and run to see the result. This is where you come to learn all the fun programming!


Each sample code will attempt to teach you something, from simple to hard, and you will have a chance to learn a simple paint program and a space game! You can get everything needed - the complete development environment and all the sample codes, by downloading the Virtual GS disk.


All the programs are developed and tested on Sweet16 emulator running on Mac OS X. They should work on other emulators or the real Apple IIGS. However, the codes might be too slow for the original or accelerated Apple IIGS.

Where do we go from here?

For object oriented programming on Apple IIGS, try Orca/Pascal.


For web programming for Safari/Firefox or iPhone/iPod touch, try GScript or GScript BASIC.


For programming on PC/Mac, try ActionScript 3.
